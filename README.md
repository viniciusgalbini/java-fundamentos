# Java Fundamentos

Repositório de estudos de fundamentos da linguagem Java e lógica de programação.

## Conteúdo
- Estrutura sequencial
- Estrutura condicional (if, else, else if)
- Entrada e saída de dados
- Operações matemáticas
- Exercícios práticos resolvidos

## Organização
Os exercícios estão separados por assunto em pastas, com prefixos nos arquivos para evitar conflito de classes.

## Objetivo
Consolidar a base em Java para evolução em back-end com Spring Boot.
